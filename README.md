# Telemedicine-App
The Telemedicine System with AI Symptom Checker enables patients to book appointments, view prescriptions, and connect with doctors through a single platform.


Telemedicine Desktop App - Prototype Version
============================================

Setup Instructions:

1. Install MySQL and create the database:
   - Run the `setup.sql` script in your MySQL client.

2. Python Dependencies:
   pip install mysql-connector-python

3. Run the App:
   python main.py
